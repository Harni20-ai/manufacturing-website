# Industrial Manufacturing E-commerce Template

A premium, production-ready multi-page e-commerce website for manufacturing and industrial companies. Built with pure HTML, CSS, and JavaScript.

## Features
- **Industrial Aesthetic**: Bold dark theme with high-contrast accents.
- **Full E-commerce Suite**: Shop, product pages, cart, and checkout.
- **Dual Dashboards**: Dedicated Admin and User dashboards.
- **Responsive Design**: Optimized for mobile, tablet, and desktop.
- **RTL Support**: Built-in support for Right-to-Left languages.
- **SEO Optimized**: Semantic HTML and meta tags included.

## Structure
- `/assets`: CSS, JS, and image assets.
- `/pages`: All HTML pages.
- `/documentation`: Detailed guides for customization.

## Installation
Simply open `pages/index.html` in any modern web browser.
